Just Readme 
            Bootstrap uses "div" tags to create layout containers, with the "row" and "col-(typ)-(width)" classes,
            where "(typ)" is xs, sm, md or large for xtra-small, small, medium & large.
            "(width)" needs to add to 12 for all columns for a row.
            e.g. col-sm-3 col-sm-3 col-sm-3 col-sm-3  (or) col-xs-6 col-xs-6 (or)col-md-9 col-md-3
            
            The rows MUST be within div class named container
            <div class="container">
                <div class="row">
                    <div class="col-md-3">
                    </div>
                    <div class="col-md-6">
                    </div>
                    <div class="col-md-3">
                    </div>
                </div>
            </div>
